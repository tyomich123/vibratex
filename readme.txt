1. Installation

- Navigate to "Appearance" -> "Add New Themes" -> "Upload Theme" -> "Choose file" (choose vibratex.zip) > Install 
- Click "Activate" after the installation
- Install all required plugin in "Dashboard" -> "Plugins" menu.
